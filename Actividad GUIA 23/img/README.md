# Carpeta de Imágenes

Esta carpeta contiene las imágenes de los disfraces subidas por los administradores.

## Permisos necesarios

Asegúrate de que esta carpeta tenga permisos de escritura para que PHP pueda guardar las imágenes subidas.

### En Linux/Mac:
```bash
chmod 755 img/
```

### En Windows:
Normalmente no es necesario cambiar permisos en XAMPP.

## Formatos soportados

- JPG/JPEG
- PNG
- GIF
- WebP
